HOW TO USE:

1. Download & Install python: https://www.python.org/downloads/

2. Create Serp API account in order to get your API KEY: https://serpapi.com/ -> https://serpapi.com/manage-api-key

3. Open serp-scrapper.py and add at line 6 your API KEY from step 2

4. Run command: pip install google-search-results (might work with pip3 instead of pip)

5. Run commnad: python serp-scrapper.py (might work with python3 instead of python)

6. See results in csv file

7. Adapt locations and keywords at lines 27/ 28 in script and play with it
